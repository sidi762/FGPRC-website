Setting up FGTracker Database parameters
1. rename include/db_connect.example.php to include/db_connect.php
2. Edit include/db_connect.php according to your server's parameters

Setting up FGTracker Web parameters
1. rename include/config.example.php to include/config.php
2. Edit include/config.php 

Setting up FGTracker interface parameters
1. rename interface.files/config.example.php to interface.files/config.php
2. Edit interface.files/config.php
